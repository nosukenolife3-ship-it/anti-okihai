# 置き配と防犯カメラの重要性

日本では最近「置き配」が増えていますが、盗難やトラブルのリスクもあります。  
アメリカでは多くの家庭が **防犯カメラ** を導入しており、被害を防いでいます。

---

## 防犯カメラを設置するメリット
1. 盗難防止
2. 配達員や訪問者の確認
3. 証拠映像の保存
